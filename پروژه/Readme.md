# To-Do List – مدیریت لیست کارهای روزانه

یک برنامه ساده کنسولی برای مدیریت تسک‌ها (کارها) نوشته شده با **C++** و تمرکز بر مفاهیم **شیءگرایی**.

## ویژگی‌ها

- افزودن تسک جدید (عنوان + مهلت + اولویت)
- نمایش همه تسک‌ها
- نمایش تسک‌های انجام‌شده / انجام‌نشده
- علامت زدن تسک به عنوان انجام‌شده
- حذف تسک از لیست
- استفاده از اولویت‌بندی (کم، متوسط، زیاد)
- مدیریت حافظه دستی با new/delete

## مفاهیم شیءگرایی استفاده شده

- **وراثت** (Inheritance) → کلاس پایه `Task` و کلاس فرزند `Todo`
- **چندریختی** (Polymorphism) → متد مجازی `show()`
- **کپسوله‌سازی** (Encapsulation) → متغیرهای خصوصی در `TodoList`
- **استفاده از STL** → `std::vector` برای ذخیره پویای تسک‌ها
- **مدیریت حافظه** → استفاده از اشاره‌گر و destructor مجازی
